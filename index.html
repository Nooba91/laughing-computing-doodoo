Case Study on Wolfram Image Identification Project and Mobilenet

Test number 1.
Output on Wolfram Image Identification Project
love seat

Output on Mobilenet
loudspeaker

Result - love seat

Wolfram Image Identification Project is correct

Test number 2.
Output on Wolfram Image Identification Project
tennis ball

Output on Mobilenet
tennis ball

Result - tennis ball

Both are correct

Test number 3.
Output on Wolfram Image Identification Project
sconce

Output on Mobilenet
microphone or mike

Result - table lamp

Wolfram is more accurate

As you have seen, Wolfram was correct on the first card, both were correct on the second,
and Wolfram was not correct, but more accurate on the third and last card. So Wolfram is the best image identifier of all.
